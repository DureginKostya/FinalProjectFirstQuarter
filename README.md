## Описание текста кода в файле Program.cs

1. Строки 1-9: Постановка задачи с примерами.
2. Строка 10: Очистка экрана каждый раз при запуске программы.
3. Строки 11-24: Метод GetNumberOfLines с входными и выходными параметрами.  
   Описание метода **GetNumberOfLines** - запрашивает у пользователя количество строк, которое он планирует ввести в массив.

   >* На входе метод получает сообщение ("Введите количество строк, которое хотите ввести в массив -") из основного кода программы (строка 69), которое далее метод отправляет на экран для пользователя.
   >* Пользователь вводит количество строк.
   >* Полученное от пользователя значение метод проверяет на правильность ввода.
   >* Если введено допустимое значение, метод возвращает его в основной код программы (строка 69). Иначе, метод рекурсивно обращается сам к себе с просьбой пользователя ввести допустимое значение. Это процедура будет повторятся до тех пор, пока пользователь не введет допустимое значение.

4. Строки 25-35: Метод GetLinesFromUser с входными и выходными параметрами.  
   Описание метода **GetLinesFromUser** - запрашивает у пользователя строки для строчного массива.

   >* На входе метод получает сообщение ("Введите строку № ") и количество строк, которое пользователь желает ввести, из основного кода программы (строка 70).
   >* Объявление строкового массива, который будет заполнен пользовательскими значениями.
   >* Организация цикла for. В цикле происходит поэтапное заполнение строкового массива, каждой ячейке массива присваивается строка, которую ввел пользователь с клавиатуры. Строки вводятся по отдельности.
   >* Метод возвращает в основной код программы (строка 70) сформированный строковый массив.

5. Строки 36-41: Метод ColorizeText только с входными параметрами.  
   Описание метода **ColorizeText** - окрашивает выводимый на экран текст.

   >* На входе метод получает сообщение (текст, который нужно окрасить) и значение, в какой цвет нужно окрасить.
   >* Установка параметров цветового вывода.
   >* Вывод текста на экран.
   >* Сброс установленных параметров цветового вывода.

6. Строки 42-54: Метод DeletingLinesFromArray с входными и выходными параметрами.  
   Описание метода **DeletingLinesFromArray** - формирует новый строковый массив из строк длиной не более 3 символов.

   >* На входе метод получает сформированный строковый массив из основного кода программы (строка 75). Массив сформирован с помощью метода GetLinesFromUser.
   >* Объявление нового строкового массива.
   >* Присвоение счетчику значения ноль.
   >* Организация цикла for. В цикле происходит поэтапная проверка длин строк в исходном строковом массиве. Если длина строки не более трех символов, то значение этой строки присваивают ячейке нового строкового массива (индекс ячейки соответствует значению счетчика), далее счетчик увеличивается на единицу. Если длина строки более трех символов, то ничего не делается.
   >* Метод возвращает в основной код программы новый строковый массив и количество строк (с длиной не более 3 символов) в массиве (строка 75).

7. Строки 55-68: Метод PrintOfArray только с входными параметрами.
   Описание метода **PrintOfArray** - выводит строковый массив на экран.  

   >* На входе метод получает строковый массив, который нужно вывести на экран, сообщение (уточняющее, выводится перед массивом), количество строк в массиве, значение цвета, в который будет окрашен вывод на экране. Все параметры получаются из основного кода программы (строка 71 и строка 76).
   >* Вывод сообщения со скобкой "[".
   >* Организация цикла for. В цикле происходит поэтапный вывод значений строкового массива. Для лучшего вывода на экран строкового массива, в цикле организовано два варианта вывода через условие. Если выводиться не последний элемент строкового массива, то реализуется первый вариант вывода, иначе - второй.
   >* Вывод скобки "]".

8. Строки 69-76: Основной код.  
   Описание основного кода.

   > **строка 69** - получение количество строк из метода GetNumberOfLines.  
   > **строка 70** - получение строкового массива из метода GetLinesFromUser.  
   > **строка 71** - вывод на экран строкового массива полученного из метода GetLinesFromUser.  
   > **строка 72** - вывод на экран поясняющего сообщения.  
   > **строки 73,74** - вывод пустых строк на экран (для красоты).  
   > **строка 75** - получение нового строкового массива (со строками с длиной не более 3 символов) и количества строк в этом массиве из метода DeletingLinesFromArray.  
   > **строка 76** - вывод на экран нового строкового массива полученного из метода DeletingLinesFromArray.
